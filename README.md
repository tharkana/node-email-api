# node-email-api

Usage
====

1. Configure 
    Copy and renam `.env.example` to `env`. And add **SENDGRID_API_KEY** by overriding *test* with your [Send Grid API Key](https://sendgrid.com/)

 2. Run server 

    ```bash
    npm i
    npm run dev
    ```
